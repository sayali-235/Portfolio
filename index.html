My Projects

Emp Info Control

EmpInfo Control is a Java-based employee management system that enables employers to securely manage employee records with CRUD operations (Create, Read, Update, Delete). Built using JSP, JDBC, MySQL, and exception handling, the system includes login and registration features for secure access. Employers can efficiently add, update, view, and delete employee details, ensuring streamlined workforce management.

Gramin Business

Gramin Business is an online marketplace that connects vendors and consumers for buying and selling soil and wooden products. Built using HTML, CSS, Java, JSP, JDBC, and MySQL, the system features three user roles—admin, vendor, and consumer. Vendors can add products with images and details, while consumers can browse, order, or cancel purchases. Admins manage user data, ensuring a smooth and direct trade experience without intermediaries.

TEXTIFY-The Text Translator

Smart Text Converter is an AI-powered system that extracts text from images using OCR technology for both printed and handwritten text. It integrates speech recognition for accurate spoken word transcription and speech-to-text conversion for seamless text generation. Built with Python and language translation APIs, it allows users to convert extracted or transcribed text into multiple languages, enhancing accessibility and communication.
